### インストール　###

クローンした後にディレクトリ移動して `pip3 install .`
削除したい場合は`pip3 uninstall pybration`

### 開発必須環境 ###

* Python3 -> brewから入れられるよ！！　Lab01のPCでも実行確認済み
* OSX version =< high sierra対応しちゃってました。
